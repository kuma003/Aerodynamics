\section{はじめに}
Barrowman Method\cite{Barrowman}は，ロケットの空力解析を簡便に行うための解析的・経験的手法である．
また，同様の手法としてはOpenRocketのドキュメント\cite{OpenRocket}も詳しく，Barrowman Methodを中心に据えて種々の補正を取り入れている．
本稿は，両者を中心にロケットの空力諸元の導出をまとめるものである．

以下，式番号において，Barrowman\cite{Barrowman}と同一のものは(B.0.0)，OpenRocket\cite{OpenRocket}と同一のものは(OR.0.0)と表記する．

\subsection{Barrowman Method}
Barrowman Methodにおいては，以下の仮定がなされていることに注意が必要である:
\begin{itemize}
  \item 迎角は非常に小さい
  \item 流れは定常 (steady) かつ渦なし (irrotational) (つまりポテンシャル流 (potential flow))
  \item 機体は剛体
  \item ノーズコーンの先端は尖っている
\end{itemize}
OpenRocketでは特にモデルロケットのようなケースを想定しているため，落下時のような迎角が大きい場合についても言及されているが，
本稿ではこの仮定の下で議論を進める．

\section{法線力係数傾斜・圧力中心について}
\section{ノーズコーン/ボディ/テールの場合}
ノーズコーンやボディ，テールは軸対称であるため本質的には同様に扱うことができる．

ここでは議論が比較的見通しやすいBarrowmanの方を中心に説明する．

\subsubsection{亜音速において}
細長く軸対称な物体においては，単位長さあたり揚力は以下のように表される：
\begin{equation}
  n(x) = \rho V \frac{\partial}{\partial x} \left[A(x)w(x)\right]
  \tag{B.3.57}
  \label{eq:barrowman_lift}
\end{equation}
ここで，$A(x)$は断面積，$w(x)$は機軸に対して垂直な方向の速度成分で，位置$x$には依存せず，
\begin{equation*}
  w(x) = V \sin\alpha \approx V\alpha.
\end{equation*}
したがって，
\begin{equation}
  n(x) = \rho V^2 \alpha \frac{\diff A(x)}{\diff x}
  \tag{B.3.59}
\end{equation} 
となる．
これからただちに径が変化しない\textbf{ボディ部にはたらく揚力は0となる}ことが分かる．

機体径は滑らかに変化するという仮定の下に，これを全体にわたって積分すると法線力$N$は\footnote{ここからBarrowmanの議論とは少し異なるが本質的に同じである.},
\begin{equation}
  N = \int_0^{l } n(x) \,\diff x =  \rho V^2 \alpha [A(l) - A(0)]
\end{equation}
となる (ここで$l$は対象とするコンポーネント長さ)．
したがって，法線力係数は，
\begin{equation}
  C_{N} = \frac{N}{\frac{1}{2}\rho V^2 A_\mathrm{ref}} = 2\frac{A(l) - A(0)}{A_\mathrm{ref}} \alpha
  % \tag{3--60\cite{Barrowman}}
\end{equation}
例えば，ノーズコーンなら，$A(0) = 0, A(l) = A_\mathrm{ref}$より，
\begin{equation}
  C_{N, \mathrm{nose}} = 2\alpha\footnotemark
\end{equation}
\footnotetext{なお，この結果は複素ポテンシャルによって計算された玉木\cite{tamaki} 式$(5\cdot 6)$の結果とも一致する．}
テールコーンなら，$A(0) = A_\mathrm{ref}, A(l) = 0$より，
\begin{equation}
  C_{N, \mathrm{tail}} = - 2 \left(1 - \frac{A(l)}{A_\mathrm{ref}}\right) \alpha
\end{equation}
となる．

法線力係数傾斜についてはこれより，
\begin{equation}
  C_{N\alpha} = \left.\frac{\partial C_{N}}{\partial \alpha}\right|_{\alpha \to 0} = 2\frac{A(l) - A(0)}{A_\mathrm{ref}}
  \tag{B.3.65}
\end{equation}
つまり，ノーズ法線力係数傾斜は，
\begin{equation}
  C_{N\alpha, \mathrm{nose}}  = 2
\end{equation}
テール法線力係数傾斜については，
\begin{equation}
  C_{N\alpha, \mathrm{tail}} = - 2 \left(1 - \frac{A(l)}{A_\mathrm{ref}}\right)
\end{equation}
となる．
ここで，このことから，\textbf{テールが生み出す法線力は負である}ことがわかる．

また，圧力中心位置については，
\begin{equation}
  X_\mathrm{cp} = \frac{\int_0^{l } x n(x) \,\diff x}{\int_0^{l } n(x) \,\diff x}
  = \frac{[x A(x)]_0^{l } - \int_0^{l } A(x) \,\diff x}{[A(x)]_0^{l }}
  = \frac{l  A(l ) - \mathrm{Vol}}{A(l ) - A(0)}
  \tag{OP.3.28}
\end{equation}
ここで，$\mathrm{Vol} = \int_0^{l } A(x)\,\diff x$はコンポーネント体積であって，
ここでは式の形をOpenRocketに合わせている．

つまり，このことから機体全長に対するノーズコーンの圧力中心位置は，
\begin{equation}
  \frac{X_{cp, \mathrm{nose}}}{l } = 1 - \frac{\mathrm{Vol}}{A(l ) l }\footnotemark
\end{equation}
\footnotetext{この表式は玉木\ref{tamaki}式$(5\cdot 7)$に合わせた．}
のように表される(テールコーンについてはほぼ同じなので省略)．

\subsection{フィンの場合}

ここでは，OpenRocketを参考に説明する．

\enskip

1枚のフィンが生み出す法線力係数傾斜はDiederichの半経験的式によって与えられる:
\begin{equation}
  (C_{N_\alpha})_1 = \frac{C_{N_{\alpha0}}F_\mathrm{D} (\frac{A_\mathrm{fin}}{A_\mathrm{ref}}\cos\Gamma_\mathrm{c})}{2 + F_\mathrm{D} \sqrt{1 + \frac{4}{F_\mathrm{D}^2}}}
  \tag{OR.3.37}
\end{equation}
ここで，
\begin{itemize}
  \item $C_{N_{\alpha0}}$: 2次元平板の法線力係数
  \item $F_\mathrm{D}$: Diederichの形状補正係数
  \item $A_\mathrm{fin}$: フィン面積
  \item $\Gamma_\mathrm{c}$: 中点の角度 (\cref{fig:fin_geometry}参照)
\end{itemize}
\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.4\linewidth]{Barrowman/img/fin_geometry.pdf}
  \caption{フィンの幾何学的パラメータ．}
  \label{fig:fin_geometry}
\end{figure}

ポテンシャル流から求められる2次元平板の法線力係数$2\pi$に，圧縮性流体に対する補正 (Prandtl-Glauert補正) を施して，
\begin{equation}
  C_{N_{\alpha0}} = \frac{2\pi}{\sqrt{1 - M^2}} = \frac{2\pi}{\beta}
  \tag{OR.3.38}
\end{equation}
である．

また，Diederichの形状補正係数$F_\mathrm{D}$は，次で与えられる:
\begin{equation}
  F_\mathrm{D} = \frac{\AR}{\frac{1}{2}C_{N_{\alpha0}}\cos\Gamma_\mathrm{c}}
  \tag{OR.3.39}
\end{equation}
ここで，$\AR$はフィンのアスペクト比であって，$\AR \equiv 2s^2/A_\mathrm{fin}$である．

以上の式を用いて整理すると，
\begin{equation}
  (C_{N_\alpha})_1 = \frac{2\pi \frac{s^2}{A_\mathrm{ref}}}{1 + \sqrt{1 + \left(\frac{\beta s^2}{A_\mathrm{fin}\cos\Gamma_\mathrm{c}}\right)^2}}
  \tag{OR.3.40}
\end{equation}

なお，F.T.E.の審査書でよく使う形式では，$A_\mathrm{fin} = \pi d^2/4$, $s/\cos\Gamma_\mathrm{c} = L_\mathrm{mc}$ (中点長. \cref{fig:fin_geometry}参照) ，
$A_\mathrm{fin} = (L_\mathrm{t} + L_\mathrm{r})s/2$ (台形フィンの場合) として，
\begin{equation}
  (C_{N_\alpha})_1 = \frac{8 \left(\frac{L_\mathrm{mc}}{d}\right)^2}{1 + \sqrt{1 + \left(\frac{2 L_\mathrm{mc}}{(L_\mathrm{t} + L_\mathrm{r})}\right)^2}}
  \tag{OR.3.41}
\end{equation}

次に，フィンが$N$枚ある場合を考える．
\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.35\linewidth]{Barrowman/img/fin_backword.pdf}
  \caption{フィンの迎角と横風成分．}
  \label{fig:fin_backword}
\end{figure}
このとき，\cref{fig:fin_backword}に示すように，横風に対してそれぞれの迎角は$\alpha_i = \alpha\sin\Lambda_i$となるから，法線力係数$(C_{N_\alpha})_{\Lambda_i}$は，
\begin{equation}
  (C_{N_\alpha})_{\Lambda_i} = (C_{N_\alpha})_1 \sin^2\Lambda_i
  \tag{OR.3.51}
\end{equation}
となる (横風に対して垂直な法線方向の力を考えるので，$\sin\Lambda_i$がさらにかけられていることに注意)．

したがって，$n$枚のフィン全体での法線力係数傾斜は，
\begin{equation}
  (C_{N_\alpha})_N = \sum_{i=1}^{n} (C_{N_\alpha})_{\Lambda_i} = (C_{N_\alpha})_1 \sum_{i=1}^{n} \sin^2\Lambda_i
  \tag{OR.3.52}
\end{equation}

ここで，$\sin^2$の和について，
\begin{equation}
  \sum_{i=1}^{n} \sin^2\Lambda_i = \frac{n}{2}
  \tag{OR.3.53}
\end{equation}
が成り立つので，最終的にフィン全体の法線力係数傾斜は，
\begin{equation}
  (C_{N_\alpha})_N = \frac{n}{2}(C_{N_\alpha})_1
\end{equation}
となる．

なお，多数のフィン ($n > 4$) の場合はフィン同士の干渉によって必ずしも$n/2$のファクターで法線力は増加しない．
詳細についてはOpenRocket\cite{OpenRocket}を参照されたい．

\enskip

最後に，フィン--ボディ間の干渉効果を考慮する．
これは，以下のような形で近似される:
\begin{equation}
  (C_{N_\alpha})_{T(B)} = K_{T(B)} (C_{N_\alpha})_N = \left(1 + \frac{r}{s + r}\right) (C_{N_\alpha})_N
  \tag{OR.3.56}
\end{equation}
もちろん，1枚フィンあたりの法線力係数傾斜$(C_{N_\alpha})_1$についても同様に補正される．


\enskip

次に，圧力中心について考える．

台形フィンの場合，機軸方向の圧力中心は，
MAC (Mean Aerodynamic Chord) 上の前方から$25\%$の位置にあるとされる．
したがって，
\begin{equation}
  X_{\mathrm{cp, fin}} = \frac{X_\mathrm{t}}{3}\frac{C_\mathrm{r} + 2C_\mathrm{t}}{C_\mathrm{r} + C_\mathrm{t}} + \frac{1}{6}\frac{C_\mathrm{r}^2 + C_\mathrm{t}^2 + C_\mathrm{r}C_\mathrm{t}}{C_\mathrm{r} + C_\mathrm{t}}
  \tag{OR.3.34}
\end{equation}
これは，フィン根本前端位置からの距離であるので，機体先端からの距離に変換するには，フィン根本前端位置を加える必要があることに注意されたい．

\section{抗力係数 (軸力係数) について}

流体中を移動する物体にかかる抗力には，粘性による抗力 (\textbf{摩擦抗力}) と，物体周りの流れの変化に伴って生じる抗力 (\textbf{圧力抗力}) がある
\footnote{そのほかにも衝撃波による抗力 (波動抗力) があるが，Barrowman Methodでは圧力抗力として扱う．が，結局本稿では亜音速域をメインに議論するので，ここでは関係ない．
また，ランチラグ等の張り出しによる寄生的な抗力 (Parastic Drag) もあるが，ここでは計算しない．
}
．
Barrowman Methodでは，これらを分けて考慮する．

なお，抗力係数に関してはほぼほぼBarrowman MethodとOpenRocketで同じであるため，ここではOpenRocketを参考に説明する．

\subsection{摩擦抗力について}
先ず初めにReynolds数について確認しよう．
Reynolds数は以下で定義される:
\begin{equation}
  R = \frac{V L}{\nu}
\end{equation}
ここで，$V$は物体速度，$L$は代表長さ，$\nu$は動粘性係数であって，大気中では$\nu \approx 1.5 \times 10^{-5} \,\mathrm{m^2/s}$である．

Reynolds数は，流れが層流か乱流かを決定する重要な無次元数である．
あるReynolds数$R_\mathrm{crit}$を超えると，境界層が乱流化し，摩擦抗力が増加する．
これは，以下で与えられる:
\begin{equation}
  R_\mathrm{crit} = 51 \left(\frac{R_s}{L}\right)^{-1.039}
  \tag{OR.3.79}
\end{equation}
ここで，$R_s$は表面の粗さである．
典型的な値を以下に示す:
\begin{table}[htbp]
  \centering
  \caption{典型的な表面粗さ}
  \label{tab:surface_roughness}
  \begin{tabular}{lc}
    \hline
    表面の種類 & 高さ [$\mu$m] \\
    \hline\hline
    平均的なガラス & 0.1 \\
    仕上げ・研磨面 & 0.5 \\
    最適なスプレー塗装面 & 5 \\
    かんなで仕上げた木製板 & 15 \\
    航空機量産塗装 & 20 \\
    滑らかなセメント面 & 50 \\
    電気亜鉛メッキ金属面 & 150 \\
    不正確なスプレー塗装された航空機塗装 & 200 \\
    粗い木製板 & 500 \\
    平均的なコンクリート面 & 1000 \\
    \hline
  \end{tabular}
\end{table}

\enskip

この値よりも小さい場合，表面摩擦係数 (Skin Friction Coeff.) $C_f$は，以下で与えられる:
\begin{equation}
  C_f = \frac{1}{1.50 (\ln R - 5.6)^2}
  \tag{OR.3.78}
\end{equation}

一方で，これよりも大きい場合には，Reynolds数には依存しなくなり，表面の粗さに依存する形で以下で与えられる:
\begin{equation}
  C_f = 0.032 \left(\frac{R_s}{L}\right)^{0.2}
  \tag{OR.3.80}
\end{equation}

また，十分小さい ($R < 10^4$) 場合には，おおむね$C_f = 1.48\times 10^{-2}$で一定値となる．

以上をまとめると，表面摩擦係数$C_f$は以下のように与えられる:
\begin{equation}
  C_f = \begin{cases}
    1.48\times 10^{-2} & \text{if\,} R < 10^4 \\
    \frac{1}{1.50 (\ln R - 5.6)^2} &  \text{if\,} 10^4 < R < R_\mathrm{crit} \\
    0.032 \left(\frac{R_s}{L}\right)^{0.2} & \text{if\,} R > R_\mathrm{crit}
  \end{cases}
  \tag{OR.3.81}
\end{equation}

\enskip

さらにこれから．圧縮性流体の補正を行う．

亜音速域なら，以下のように補正される:
\begin{equation}
  C_{f_c} = C_f (1 - 0.1 M^2)
  \tag{OR.3.82}
\end{equation}

超音速域なら，以下のように補正される:
\begin{equation}
  C_{f_c} = \frac{C_f}{(1 + 0.15 M^2)^{0.58}}
  \tag{OR.3.83}
\end{equation}

\enskip

以上から，ノーズの摩擦抗力係数は以下で与えられる:
\begin{equation}
  (C_{D, \text{nose}})_\text{friction} = C_{f_c} \frac{A_\mathrm{wet, nose}}{A_\mathrm{ref}}
\end{equation}
ここで$A_\mathrm{wet, nose}$はノーズコーンの濡れ面積 (表面積) である．

また，円筒形の場合は，細長比$f_B = L / r$ (半径を使うことに注意) によってさらに補正され．
最終的な摩擦抗力係数は以下のようになる:
\begin{equation}
  (C_{D, \text{body}})_\text{friction} = C_{f_c} \left(1 + \frac{1}{2 f_B}\right) \frac{A_\mathrm{wet, body}}{A_\mathrm{ref}}
\end{equation}
ここで$A_\mathrm{wet, body}$はボディの濡れ面積である．
テールのような窄められた形状でも同様であるが，濡れ面積が変わることに注意が必要である．

フィンの場合は，MACの長さ$\bar{c}$と厚み$t$を用いて，以下で与えられる:
\begin{equation}
  (C_{D, \text{fin}})_\text{friction} = C_{f_c} \left(1 + \frac{2t}{\bar{c}}\right) \frac{A_\mathrm{wet, fin}}{A_\mathrm{ref}}
\end{equation}
ここで$A_\mathrm{wet, fin}$はフィンの濡れ面積である．
フィンの場合は，枚数と裏表によって単純なフィンの面積から$2n$倍されることに注意されたい．

\subsection{圧力抗力について}
圧力抗力は剥離流れ等によって生じるもので，ポテンシャル流では説明できない．
そこでより経験的な式が用いられる．


ノーズコーンの場合，以下のように与えられる:
\begin{equation}
  (C_{D, \text{nose}})_p = 0.8 \sin^2 \phi
  \tag{OR.3.86}
\end{equation}
ここで，$\phi$はノーズとボディの接合部がなす角である．
実際にはここから圧縮性の補正が入るが，もはやノーズ形状等に依存する形となってしまうので，ここでは扱わない
\footnote{
  私が作成した計算スプシでも，ここの計算は省略した．
  例えば円錐形ならば比較的簡単に与えられるが．OgiveやHaack形状などではより複雑になり，
  実測値から細長比の補正を施しつつ内挿・外挿する形で求める必要がある．
}．

\enskip

次にbase抗力について考える．
これは，ロケットの底部や急激に径が変化する部位で必ず生じ得るものである．
これは，以下の形で与えられる．
\begin{equation}
  (C_{D})_\mathrm{base} = 
  \begin{cases}
    0.12 + 0.13 M^2 & \text{if\, } M < 1\\
    0.25 / M & \text{if\, } M > 1
  \end{cases}
  \tag{OR.3.94}
\end{equation}
テールがない場合，これがそのままボディ部の圧力抗力係数に相当する．

もしテールがある場合は，テールによって，この抗力が低減する．
テール部について$\gamma = l_\mathrm{tail}/(d_1 - d_2)$ (テール長/前方と後方直径差) とおくと，
\begin{equation}
  (C_{D, \text{body}})_p 
  \left(\frac{A_\text{tail}}{A_\mathrm{ref}}\right)^2 (C_{D})_\mathrm{base} 
  \begin{cases}
    1 & \text{if\, } \gamma < 1 \\
    (3 - \gamma)/2 & \text{if\, } 1 < \gamma < 3 \\
    0 & \text{if\, } \gamma > 3
  \end{cases}
  \tag{OR.3.88}
\end{equation}
となる (ここで$A_\text{tail}$はテール断面積)
\footnote{
  原典では，最初の係数は$A_\text{base}/A_\text{boattail}$となっている．
  それぞれ後方，前方の断面積であるので，この場合 $A_\text{tail} / A_\text{ref}$と等価である．
  最終的にこの値について，参照断面積で正規化する過程で$A_\text{tail} / A_\text{ref}$がかけられるため，一致する．
}．

\enskip

フィンの場合は複雑である．
ここでは，台形フィン・前縁の端部は矩形であると仮定する．

先ず初めに，前縁部について，前縁部に垂直な方向の抗力は，
\begin{equation}
  (C_{D})_{\text{LE} \perp} = (C_{D})_\mathrm{stag}.
  \tag{OR.3.90}
\end{equation}
ここで，$(C_{D})_\mathrm{stag}$は鈍頭な円柱における抗力係数であって，
よどみ圧を正規化した値として，
\begin{equation}
  \frac{q_\text{stag}}{q} = \begin{cases}
    1 + \frac{M^2}{4} + \frac{M^4}{40} & \text{if\, } M < 1 \\
    1.84 - \frac{0.76}{M^2} + \frac{0.166}{M^4} + \frac{0.035}{M^6} & \text{if\, } 1 < M \\
  \end{cases}
  \tag{OR.B.1}
\end{equation}
を用いて，
\begin{equation}
  (C_{D})_\mathrm{stag} = 0.85\frac{q_\text{stag}}{q}
  \tag{OR.B.2}
\end{equation}
で与えられる．

これを流れ方向に補正するには，フィンの前縁角$\Gamma_\text{L}$を用いて，
\begin{equation}
  (C_{D})_{\text{LE}} = (C_{D})_{\text{LE} \perp} \cos^2 \Gamma_\text{L}
  \tag{OR.3.91}
\end{equation}
とすればよい．

次にフィン後端の抗力については，端部を丸めていない場合にはbase抗力にほかならず，
\begin{equation}
  (C_{D})_{\text{TE}} = (C_{D})_\mathrm{base}
  \tag{OR.3.92}
\end{equation}
で与えられる．

従って，フィン全体の圧力抗力係数は，
\begin{equation}
  (C_{D, \text{fin}})_p = \left((C_{D})_{\text{LE}} + (C_{D})_{\text{TE}}\right) \frac{A_\mathrm{fin, front}}{A_\mathrm{ref}}
  \tag{OR.3.93}
\end{equation}
となる．ここで，$A_\mathrm{fin, front}$はフィンの前縁部の投影断面積であって，$A_\mathrm{fin, front} = N\cdot ts$で与えられる．
